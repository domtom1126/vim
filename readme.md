# Vim Setup

```
brew install nvim
```
```
cd ~/.config
# if it doesn't exist
mkdir nvim
```
## Some Cheats
Just to make things easy I set up some aliases.

```
vim ~/.zshrc
```
```
alias vim="nvim"
alias ncon="~/.config/nvim"
```
